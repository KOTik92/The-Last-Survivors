# The Last Survivors

Роль в проекте: Front-End Unity Developer

Сборка под Андройд

Игра где предстоит управлять двумя персонажами что бы зачистить локации от зомби и собрать хлам что бы пройти дальше.
Сменить персонажа можно в любой момент игры.
